

# Drum Kit

An interactive, browser-based virtual drum kit! Create your own beats by clicking the drum pads on the screen or by using the designated keys on your keyboard.

## Preview

## Features

* **Interactive UI:** Clean, responsive design featuring distinct drum and cymbal images.
* **Dual Controls:** Playable via mouse clicks or keyboard presses.
* **Visual Feedback:** Buttons likely provide a visual animation (like a quick flash or shadow change) when pressed to simulate hitting a drum.
* **Instant Audio:** Plays the corresponding drum sound without delay.

---

## How to Play

You can play the drums by either clicking the icons with your mouse or pressing the following keys on your keyboard:

| Key | Instrument |
| --- | --- |
| **w** | Crash / Cymbal |
| **a** | Kick Drum |
| **s** | Snare |
| **d** | Tom 1 |
| **j** | Tom 2 |
| **k** | Tom 3 |
| **l** | Tom 4 |

---

## Technologies Used

* **HTML5:** To structure the layout and drum buttons.
* **CSS3:** For styling, positioning, and adding visual flair (like custom fonts and background colors).
* **JavaScript (Vanilla):** To handle event listeners for mouse clicks and keyboard strokes, and to trigger the audio files.
